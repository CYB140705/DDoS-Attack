## 🔧 功能说明

此程序是一个简单的UDP洪水攻击(DDoS)演示工具，主要用于教学目的，展示:
- 基本的socket编程
- UDP数据包发送
- 简单的用户交互界面

## 📦 依赖项

本程序使用Python标准库，无需额外安装依赖:
- `socket`
- `random`
- `time`
- `os`
- `sys`
- `datetime`

## 🚀 使用方法

1. 确保已安装Python 3环境
2. 运行程序:
   ```bash
   python ddos-windows.py(适用于Windows) / ddos-linux.py(适用于Linux/Mac)
   ```
3. 按照提示输入:
   - 目标IP地址
   - 目标端口号
   - 攻击速度(1~1000)
